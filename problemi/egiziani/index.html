Interi egiziani

Un intero \(n\) si definisce egiziano se esiste una sequenza di interi positivi strettamente crescente \(0 < a_1 < a_2 < ... < a_k = n\) (quindi l'ultimo termina della sequenza è \(n\)) tale che $$ \frac{1}{a_1} + \frac{1}{a_2} + ... + \frac{1}{a_k} = 1 $$

  1. Determinare se il numero \(n = 72\) è egiziano.
  2. Determinare se il numero \(n = 71\) è egiziano.
  3. Determinare se il numero \(n = 72^{71}\) è egiziano.
